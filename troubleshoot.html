üîç Common Issues & Solutions

1. Browser Cache Issue

Solution: Hard refresh your browser:

  • Windows: Ctrl + F5 or Ctrl + Shift + R
  • Mac: Cmd + Shift + R

2. Browser Developer Tools Check

Steps:

  1. Open your browser
  2. Go to http://localhost:5173/
  3. Press F12 to open Developer Tools
  4. Check the Console tab for any red errors
  5. Check the Network tab to see if resources are loading

3. Firewall/Antivirus Blocking

Some antivirus software blocks localhost connections.

Solution: Temporarily disable antivirus or add localhost to exceptions

4. Port Conflict

If port 5173 is busy, try port 5174:

http://localhost:5174/

üö® If Nothing Works

Complete Reset:

  1. Stop the development server (Ctrl + C in terminal)
  2. Clear npm cache: npm cache clean --force
  3. Delete node_modules: Remove-Item -Recurse -Force node_modules
  4. Reinstall: npm install
  5. Start fresh: npm run dev
